# Council of Minds — Example Debates

> Screenshots and sample outputs from real debates run on the app.
> Add your own by running a debate, taking a full-page screenshot, and dropping it in `screenshots/`.

---

## How to capture a full-page screenshot

**Chrome / Edge (easiest):**
1. Open the debate result in your browser
2. Press `F12` → open DevTools
3. Press `Ctrl+Shift+P` → type `screenshot`
4. Select **"Capture full size screenshot"**
5. Save to `examples/screenshots/`

**Firefox:**
1. Right-click anywhere on the page
2. Select **"Take Screenshot"**
3. Click **"Save full page"**
